# Hangman Game

## Description

This is a simple command-line version of the popular Hangman game, where the player has to guess letters of a hidden word. The player has a limited number of lives, and for each incorrect guess, they lose a life. The game ends when either the word is completely guessed or the player runs out of lives.

Hangman is a game made by using Python.

![Version](https://img.shields.io/badge/Version-v1.0-orange)
![Python](https://img.shields.io/badge/Python-v3.0%2B-blue)

## How to Play

1. The game will show a random word with blanks representing each letter.
2. You will be prompted to guess a letter.
3. If the letter is in the word, it will replace the corresponding blanks.
4. If the letter is not in the word, you will lose a life.
5. You have a total of 6 lives. If you run out of lives, you lose the game.
6. The game ends when you either guess all the letters correctly or lose all lives.

## How to Run

1. Clone or download the repository.
2. Install Python 3.x on your machine (if not already installed).
3. Navigate to the project directory.
4. Ensure you have the `hangman_words.py` file containing the list of words and `hangman_art.py` for the visual hangman stages.
5. Run the game by executing the following command in your terminal:

    ```bash
    python hangman_game.py
    ```

6. The game will start and you can follow the prompts to guess the letters.
