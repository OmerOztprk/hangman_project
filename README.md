Hangman Game  

A simple command-line Hangman game written in Python. Try to guess the hidden word before you run out of lives!

Game Rules
- A random word is chosen at the start of the game.
- The player guesses one letter at a time.
- Correct guesses reveal the letter in the word.
- Incorrect guesses reduce the number of lives and build the hangman figure.
- The game ends when the word is fully guessed (win) or when all lives are lost (lose).
- After the game ends, the player can choose to play again.

Features  
- Random word selection  
- User input validation (only letters allowed)  
- Keeps track of guessed letters  
- ASCII art representation of Hangman  
- Win/loss conditions with replay option
